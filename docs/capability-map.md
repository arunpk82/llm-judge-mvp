# 4. Capability Domains

The platform capabilities fall into five domains.

Evaluation

• rule engine
• rubric scoring
• evaluation matrix

Governance

• dataset registry
• baseline promotion
• rule lifecycle

Observability

• run registry
• evaluation telemetry
• drift detection

Adjudication

• hybrid LLM judges
• confidence calibration
• human escalation

Enterprise Platform

• RBAC
• multi‑tenant architecture
• compliance reporting

# 5. Strategic Capability Pillars

The long‑term platform strategy is built around four pillars.

Evaluation Infrastructure

Deterministic evaluation framework.

Governance Platform

Policy enforcement and compliance support.

AI Safety Layer

Continuous monitoring and adversarial testing.

Decision Intelligence

Insights from evaluation data.
